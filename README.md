Проект: Анализ резюме из HeadHunter
Описание проекта
Проект представляет собой комплексный анализ данных с платформы HeadHunter, включающий преобразование сырых данных, исследование зависимостей и очистку данных для дальнейшего моделирования.

Основные этапы анализа
1. Преобразование данных
Образование: Выделение уровня образования из текстового поля

Демография: Разделение признака "Пол, возраст" на отдельные категории

Опыт работы: Конвертация опыта работы в месяцах с обработкой различных форматов

География: Классификация городов и анализ мобильности соискателей

Занятость: One-Hot Encoding для признаков занятости и графика работы

Зарплата: Конвертация валют в рубли с использованием исторических курсов

2. Исследование зависимостей
Распределения: Анализ возраста, опыта работы и зарплатных ожиданий

Влияние образования: Медианные зарплаты по уровням образования

Географический анализ: Различия зарплат между городами

Мобильность: Влияние готовности к переезду и командировкам

Комбинированный анализ: Тепловые карты и многоуровневые диаграммы

3. Очистка данных
Дубликаты: Удаление полных дубликатов

Пропуски: Заполнение и удаление пропущенных значений

Выбросы: Очистка аномальных зарплат и возрастов

Валидация: Проверка логической согласованности данных

Ключевые технологии
Pandas для обработки данных

Plotly и Seaborn для визуализации

NumPy для численных операций

Методы EDA для разведывательного анализа

Результаты
Проект демонстрирует полный цикл работы с данными: от первоначальной обработки сырых данных до глубокого анализа зависимостей и подготовки данных для машинного обучения. Выявлены ключевые факторы, влияющие на зарплатные ожидания соискателей, и проведена комплексная очистка данных.

Использование
Код представлен в виде Jupyter ноутбука с последовательным выполнением этапов анализа. Все графики интерактивны и сопровождаются детальными выводами.
